# 📱 WhatsApp Chat Analysis

## 📌 Project Overview
This project analyzes exported WhatsApp chat data using Python. It provides insights into user activity, messaging behavior, and chat statistics through data analysis and visualizations.

## 🚀 Features
- Total Messages
- Total Words
- Media Shared
- Links Shared
- Most Active User
- Daily Timeline
- Monthly Timeline
- Activity Heatmap
- Emoji Analysis
- Word Cloud
- Most Common Words

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- WordCloud
- Emoji
- URLExtract

## 📂 Project Structure

## 📊 Visualizations
- Bar Charts
- Line Charts
- Heatmaps
- Word Cloud
- Pie Charts

## ▶️ How to Run

1. Clone the repository
2. Install the required libraries

```bash
pip install -r requirements.txt
```

3. Run the notebook

```
jupyter notebook first_minor_project.ipynb
```

## 📈 Results
This project helps understand communication patterns, user engagement, emoji usage, and message trends in WhatsApp chats.
